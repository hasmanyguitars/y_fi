# YFi
This is a Ruby implementation for fetching stock/fund prices from the Yahoo Finance API with YQL: https://developer.yahoo.com/yql/faq/

## Usage

## Installation
`gem install y_fi`

```ruby
gem 'y_fi'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install y_fi
```
